						File Manager 1.0
					Инструкция по работе с приложением

Для выбора нужной операции используйте клавиши "стрелка вниз" и "стрелка вверх" и клавишу "Enter".
Для отмены текущей операции нажмите "Escape".
Сверху показывается текущий адрес, в котором вы находитесь.
Для корректной работы приложения лучше развернуть консоль на весь экран.
Список доступных операций:
1)"Просмотр списка дисков компьютера и выбор диска" - выполняет поиск дисков на компьютере и дает возможность выбрать диск.
(Текущий адрес становится выбранным диском)
2)"Выбор директории" - выполняет поиск директорий по текущему адресу и дает возможность выбрать директорию.
(Текущий адрес становится выбранной директорией)
3)"Просмотр списка файлов в директории" - выполняет поиск файлов по текущему адресу и дает возможность выбрать файл.
(Текущий адрес становится выбранным файлом)
4)"Вывод содержимого текстового файла в консоль в кодировке UTF-8" - выводит содержимое текстового файла в кодировке UTF-8.
(Текущий адрес должен быть адресом выводимого файла)
5)"Вывод содержимого текстового файла в консоль в выбранной кодировке" - выводит содержимое текстового файла в выбранной пользователем кодировке.
(Текущий адрес должен быть адресом выводимого файла)
6)"Копирование файла" - создает копию файла, находящегося по текущему адресу.Требуется указать название файла вместе с форматом.
Возможно копирование не только файлов .txt, нужно указать тип исходного файла.
(Текущий адрес становится адресом копии)
7)"Перемещение файла в выбранную директорию" - Перемещает файл, находящийся по текущему адресу, в введеную пользователем директорию.Требуется указать название директории.
(Текущий адрес становится адресом, по которому переместили файл)
8)"Удаление файла" - удаляет файл, находящийся по текущему адресу.
(Текущий адрес становится пустым)
9)"Создание простого текстового файла в кодировке UTF-8" - создает файл в кодировке UTF-8 в находящейся по текущему адресу директории.Требуется написать название файла вместе с форматом и его содержимое.
Для оконачания записи введите пустую строку. Возможно создание файла не в формате .txt
(Текущий адрес становится адресом созданного файла)
10)"Создание простого текстового файла в выбранной кодировке" - создает файл в кодировке UTF-8 в находящейся по текущему адресу директории.Требуется написать название файла вместе с форматом и его содержимое.
Для оконачания записи введите пустую строку. Возможно создание файла не в формате .txt
(Текущий адрес становится адресом созданного файла)
11)"Конкатенация содержимого двух или более текстовых файлов и вывод результата в консоль в кодировке UTF - 8." - требуется указать адреса файлов.Для оконачания ввода адреса введите пустую строку.


Нажмите любую клавишу для продолжения.
